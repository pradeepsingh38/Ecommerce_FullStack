# 🛒 E-Commerce Web Application

A full-stack eCommerce web application built using **React.js (Frontend)** and **Spring Boot (Backend)**. The application provides secure authentication, product management, and a responsive user interface.

---

## 🚀 Features

- 🔐 User Authentication (JWT-based login & registration)
- 🧑 User & Admin role support
- 📦 Product Management (Add, View, Update, Delete)
- 🛍️ Browse Products
- 🔒 Protected Routes (Frontend + Backend)
- 🌐 RESTful APIs
- 🎨 Responsive UI using CSS Modules

---

## 🛠️ Tech Stack

### Frontend
- React.js
- JavaScript (ES6+)
- CSS Modules

### Backend
- Spring Boot
- Spring Security
- JWT Authentication
- Hibernate / JPA

### Database
- MySQL
